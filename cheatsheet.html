An ES5 + Cheat Sheet

Unlike other libraries, Agave doesn't replicate anything already in ES5.

Methods added by Agave are prefixed.

Starting Agave

  • av.enable(prefix)

    Pick a prefix string that uniquely defines your project or organization.

Methods

Array

  • indexOf(value)

    Returns the (first) index of an element equal to the value, or -1 if not found.

  • forEach(function)

    Run function over each item in the array.

  • map(function)

    Make a new array with the results of calling a provided function on every item.

  • filter(function)

    Creates a new array with all of the elements of this array for which the provided filtering function returns true.

  • every(function)

    Returns true if every element in this array satisfies the provided testing function.

  • reduce(function)

    Apply a function simultaneously against two values of the array (from left-to-right) as to reduce it to a single value.

  • some(function)

    Returns true if at least one element in this array satisfies the provided testing function.

  • prefixclone()

    Make a independent copy of the array.

  • prefixcontains(value)

    Return true if the value appears in the array.

  • prefixextend(newItems)

    Add items from newItems array.

Object

  • prefixforEach(function)

  • prefixgetKeys()

  • prefixgetSize()

  • prefixgetPath(path)

  • prefixclone()

  • prefixextend(newProperties)

Function

  • bind()

  • prefixthrottle()

String

  • trim()
  • prefixcontains(substring)

  • prefixstartsWith(substring)

  • prefixendsWith(substring)

  • prefixrepeat(count)

  • prefixreverse()

  • prefixforEach(function)

Number

  • x()
  • prefixx()

Agave.js safely extends JS with helpful, intuitive methods that make your code shorter and more readable.
